# Description
This project is an introduction to computer networking. The first few levels focus on calculations and understanding mask vs IP and how they're used. Next it is about configuring IP addresses and masks to fullfill a pre set list of goals.

# Instructions
1. Download the files from intra
2. Extract files as stated in the subject
3. Open 'index.html', enter your login and start
4. After each task, download the .json file through the 'Get my config' button, you can use this to check on exercises you've completed
5. Submit all ten levels (.json files) and read me file through your git repo
